Bayesiano ingenuo

La extensión del clasificador Naive Bayes se ajusta al modelo de clasificación Naive Bayes para una variable dependiente categórica. Puede especificar si se utilizan todos los predictores disponibles o si el procedimiento selecciona los mejores predictores. La extensión se puede utilizar también para la selección de predictores sin clasificación.

Variables

Variable dependiente. Especifique la variable numérica o de cadena que se debe analizar.

Método de especificación de variables. Puede especificar un conjunto de variables para excluirlo del análisis o el conjunto de factores y covariables para incluirlo en el análisis. Si no especifica ninguna variable a excluir ni ningún factor candidato o covariable, se incluyen todas las variables (excepto la variable dependiente y la variable de ponderación).

Variables a excluir. Especifique variables para excluirlas del análisis. De manera predeterminada, el procedimiento selecciona los mejores predictores de las variables restantes (la variable dependiente y la variable de ponderación siempre se excluyen).

Factores y covariables. Especifique los factores y las covariables que se deben incluir en el análisis. Los factores se tratan como variables categóricas y pueden tener valores numéricos o de cadena. Las covariables se tratan como variables de escala y deben ser numéricas. De manera predeterminada, el procedimiento selecciona los mejores predictores del conjunto especificado de factores y covariables.

Variables forzadas. Se pueden especificar factores y covariables que siempre se utilizan en el modelo de clasificación. Los factores forzados se tratan como variables categóricas y pueden tener valores numéricos o de cadena. Las covariables forzadas se tratan como variables de escala y deben ser numéricas.

Resultado

Resumen de proceso de casos. Resume el número de casos que se incluyen y excluyen del análisis.

Predictores excluidos. Ofrece una lista de los predictores que se excluyen debido a valores perdidos o constantes para todos los casos. Los predictores excluidos se muestran por tipo (categóricos o de escala).

Predictores seleccionados. Muestra los predictores seleccionados por tipo (categóricos o de escala).

Resumen de subconjuntos de predictores. Muestra un resumen de la secuencia de subconjuntos de predictores.

Clasificación de casos. Muestra el número de casos que están clasificados correcta e incorrectamente para cada categoría de variable dependiente. Si hay datos de prueba definidos, se proporcionan los resultados de la clasificación para las muestras de entrenamiento y prueba. Si no se definen datos de prueba, los resultados de la clasificación solo se proporcionan para la muestra de entrenamiento.

Guardar

Valores pronosticados y probabilidades pronosticadas. Puede guardar valores pronosticados y probabilidades pronosticadas como nuevas variables en el conjunto de datos activo. El valor pronosticado es la categoría de variable dependiente con la mayor probabilidad posterior tal como se ha estimado mediante el modelo Naive Bayes. Se guardan las probabilidades pronosticadas de las primeras N categorías de la variable dependiente, donde N es 25 de manera predeterminada.

Opciones

Valores perdidos del usuario para variables categóricas. Controla si los valores perdidos del usuario para variables categóricas se excluyen del análisis o se incluyen en él. Se excluyen siempre los valores perdidos del usuario para variables de escala. Los valores perdidos del sistema correspondientes a cualquier variable siempre se excluyen.

Muestra de entrenamiento. Puede particionar el conjunto de datos activo en muestras de entrenamiento y de prueba especificando un porcentaje de casos seleccionados aleatoriamente para asignarlos a la muestra de entrenamiento.

Selección de predictores.

• Seleccionar el mejor subconjunto de la secuencia de subconjuntos. De manera predeterminada, el procedimiento Naive Bayes selecciona el mejor subconjunto de predictores de una secuencia de subconjuntos, con un número creciente de predictores en cada subconjunto. El procedimiento determina automáticamente el tamaño del subconjunto más grande, pero también se puede especificar un tamaño de subconjunto máximo explícito. Cuando el conjunto de datos activo está particionado en muestras de entrenamiento y de prueba, se puede elegir si el mejor subconjunto se determina a partir de un criterio pseudo-BIC que se basa en la muestra de entrenamiento o en un criterio de datos de prueba que se basa en la muestra de prueba.

• Seleccionar el mejor subconjunto del tamaño especificado. Esta opción especifica el número exacto de predictores que se deben incluir en el mejor subconjunto.

• Usar todos los predictores especificados. Esta opción indica que todos los predictores especificados se incluyen en el subconjunto final y resulta útil para la creación de modelos sin selección de predictor.

Límite de tiempo. Especifica el número máximo de minutos durante el cual se puede ejecutar el procedimiento Naive Bayes. Si se supera el límite de tiempo, el procedimiento se termina y no se proporcionan resultados. Al especificar un valor de 0 se apaga el temporizador.

Máximo de memoria para almacenar datos de entrenamiento. Especifica la cantidad máxima de memoria en megabytes (MB) que el procedimiento Naive Bayes utiliza para almacenar datos de registros de entrenamiento al calcular la log-verosimilitud media. Si la cantidad de memoria que se requiere para almacenar registros es superior, en su lugar se utilizan datos simulados.

Número de intervalos para predictores de escala. Especifica el número de intervalos que se deben usar a dividir el dominio de un predictor de escala en intervalos de ancho igual.

Características adicionales

Este diálogo genera la sintaxis para el comando NAIVEBAYES. En la sintaxis de NAIVEBAYES, se puede especificar una variable que particione el conjunto de datos en muestras de entrenamiento y de prueba. Además, se puede guardar el modelo Naive Bayes en un archivo XML para poder puntuar datos nuevos con el modelo.

Para obtener más información sobre el procedimiento Naive Bayes, consulte la documentación del comando NAIVEBAYES en el sistema de ayuda de IBM SPSS Statistics.
